마트나 시장에서 장을 볼 때 신선한 식재료를 고르는 건 생각보다 어려워요. 겉으로 봐서는 다 비슷해 보이는데, 막상 집에 와서 요리하면 맛이 다르고 빨리 상하기도 하죠. 신선도 확인법을 알면 항상 좋은 식재료를 고를 수 있어요.

오늘은 채소, 과일, 육류, 해산물, 계란까지 식재료 종류별로 신선도를 확인하는 방법을 총정리해드릴게요.

## 채소 신선도 확인법

### 공통 원칙

- **잎채소**: 잎이 선명한 색이고 탄탄한 것. 시든 것, 노란 것 피하기
- **뿌리채소**: 단단하고 무거우며 흠집이 없는 것
- **수분**: 들었을 때 묵직한 게 수분이 많은 것

### 상추·시금치·깻잎

- 잎이 선명한 녹색이고 탄탄한 것
- 시들거나 끝이 노란 것은 신선도 감소
- 향이 진하게 나는 것이 좋아요

### 오이

- 전체적으로 짙은 녹색이고 표면에 가시가 있는 것
- 굽거나 끝이 시든 것 피하기
- 들었을 때 무겁고 단단한 것

### 당근

- 전체적으로 선명한 오렌지색인 것
- 표면이 매끄럽고 잔뿌리가 적은 것
- 잎이 붙어 있다면 잎이 신선한 것이 더 신선한 당근

### 브로콜리

- 꽃봉오리가 촘촘하고 진녹색인 것
- 노란 부분이 없는 것 (노랗게 변하면 신선도 감소)
- 밑동이 신선하고 촉촉한 것

### 감자·고구마

- 껍질이 매끄럽고 흠집이 없는 것
- 싹이 나지 않은 것
- 단단하고 무거운 것
- 감자: 녹색 부분이 없는 것 (녹색 부분은 독성 물질 함유)

## 과일 신선도 확인법

### 공통 원칙

- **향**: 향이 진할수록 잘 익은 것
- **무게**: 같은 크기라면 무거운 게 수분이 많음
- **꼭지**: 꼭지가 신선할수록 최근 수확된 것

### 수박

- 두드렸을 때 통통하고 울리는 소리가 나는 것
- 표면의 줄무늬가 선명하고 색 대비가 뚜렷한 것
- 꼭지가 신선하게 붙어 있는 것
- 배꼽(꼭지 반대편) 부분이 작은 것

### 사과

- 탱탱하고 윤기가 있는 것
- 흠집이 없고 색이 고른 것
- 향이 달콤하게 나는 것
- 꼭지가 싱싱한 것

### 포도

- 알이 송이에 탄탄하게 붙어 있는 것
- 알 표면에 하얀 분(과분)이 있는 것 (신선의 표시)
- 알이 골고루 탄탄한 것

### 복숭아

- 향이 달콤하게 나는 것
- 살짝 눌렸을 때 스프링 같은 탄력이 있는 것
- 솜털이 풍성하게 남아 있는 것

## 육류 신선도 확인법

육류는 신선도가 안전과 직결되어 있어요.

### 소고기

- **색깔**: 선명한 붉은색 (선홍색). 갈색이나 회색은 신선도 감소
- **냄새**: 약간의 고기 냄새는 정상, 시큼하거나 역한 냄새는 부적합
- **결**: 결이 촘촘하고 탄탄한 것
- **지방**: 지방이 흰색인 것 (노란 지방은 오래된 것)

### 돼지고기

- **색깔**: 연분홍색에서 옅은 분홍색
- **냄새**: 쿰쿰하거나 역한 냄새 없어야 함
- **수분**: 지퍼백 안에 물이 너무 많으면 오래된 것

### 닭고기

- **색깔**: 연분홍색 또는 연한 살구색
- **냄새**: 달큰한 냄새가 없는 것 (있으면 신선도 감소)
- **피부**: 손으로 눌렀다 떼면 바로 돌아오는 탄력

## 해산물 신선도 확인법

![식재료 신선도 확인의 핵심은 색·향·촉감 세 가지예요. 좋은 식재료를 고르는 능력이 요리의 기본이에요](https://images.pexels.com/photos/3987358/pexels-photo-3987358.jpeg?auto=compress&cs=tinysrgb&w=800&h=500&dpr=1)

### 생선

- **눈**: 맑고 선명한 것 (탁하면 신선도 감소)
- **아가미**: 붉은색이고 이물질 없는 것
- **비늘**: 윤기 있고 단단히 붙어 있는 것
- **냄새**: 바다 냄새 정도. 심한 비린내는 부적합
- **살**: 눌렀을 때 탄력이 있는 것

### 새우

- **색깔**: 투명하거나 회색. 붉게 변한 것은 신선도 감소
- **껍질**: 단단히 붙어 있는 것
- **냄새**: 깔끔한 바닷물 냄새

### 굴

- **광택**: 윤기가 있고 통통한 것
- **색깔**: 투명감 있는 유백색
- **냄새**: 신선한 바다 향

## 계란 신선도 확인법

### 물에 담가보기

계란 신선도를 확인하는 가장 정확한 방법이에요.

- **바닥에 누워 있으면**: 매우 신선
- **한쪽이 살짝 떠 있으면**: 신선하지만 조만간 먹어야 함
- **세로로 서면**: 신선도 낮음, 빨리 먹어요
- **완전히 뜨면**: 상한 것, 버리세요

### 흔들어보기

흔들었을 때 내부에서 소리가 나거나 찰랑거리면 신선도가 많이 떨어진 거예요. 신선한 계란은 흔들어도 소리가 없어요.

## 마무리

신선한 식재료를 고르는 능력은 요리의 출발점이에요. 색깔, 냄새, 촉감 이 세 가지를 종합적으로 확인하면 대부분의 식재료 신선도를 판단할 수 있어요.
