6월은 봄과 여름이 교차하는 시기예요. 봄 식재료는 마무리되고, 본격적인 여름 식재료들이 등장하기 시작하는 풍성한 달이에요. 매실이 익어가고, 참외가 달아지며, 복분자가 검붉게 물드는 6월. 지금 먹어야 가장 맛있는 제철 식재료를 총정리해드릴게요.

## 6월 제철 채소

### 매실

6월의 주인공이라 해도 과언이 아닌 매실. 6월 초~중순이 수확 시기예요. 청매실은 6월 초에 나오고, 황매실은 6월 중순~말에 나와요.

**청매실:** 아직 덜 익은 상태로, 청매실청이나 매실주 담그기에 적합해요.
**황매실:** 충분히 익은 것으로, 단맛이 더 강하고 향이 풍부해요. 매실잼이나 매실 발효액에 좋아요.

**고르는 법:**
- 표면에 흠집이 없고 단단한 것
- 청매실은 진한 초록색, 황매실은 노란빛이 골고루 든 것
- 향이 은은하게 나는 것

**활용:** 매실청, 매실주, 매실 장아찌, 매실잼

### 감자 (햇감자)

6월에는 햇감자가 나와요. 겨울에 저장해둔 일반 감자와 달리 햇감자는 껍질이 얇고 수분이 많아 더 촉촉하고 담백해요.

**고르는 법:**
- 껍질이 얇고 매끈한 것
- 표면에 초록색 부분이 없는 것 (솔라닌 독소 함유)
- 눌렀을 때 단단한 것
- 싹이 나지 않은 것

**활용:** 감자조림, 감자국, 감자전, 매시 포테이토, 감자볶음

### 오이

초여름 오이가 본격적으로 쏟아지는 시기예요. 6월 오이는 수분이 풍부하고 아삭함이 최고예요.

**고르는 법:**
- 짙은 초록색으로 균일한 것
- 꼭지가 싱싱하고 시들지 않은 것
- 가시가 선명하게 살아 있는 것
- 묵직한 것

**활용:** 오이소박이, 오이무침, 냉국, 오이김치, 샐러드

### 양상추

봄부터 초여름이 양상추의 제철이에요. 잎이 두껍고 아삭한 맛이 최고조에 달하는 시기예요.

**고르는 법:**
- 잎이 빽빽하게 차 있고 묵직한 것
- 잎 끝이 갈변하지 않은 것
- 단면이 촉촉하고 흰색인 것

**활용:** 샐러드, 쌈밥, 버거 속 재료, 쌈장과 함께

### 풋마늘·마늘쫑

6월은 마늘이 수확되기 직전으로, 풋마늘과 마늘쫑이 나오는 시기예요.

**마늘쫑:** 마늘꽃이 피기 전 줄기 부분으로, 아삭하고 향이 진해요.
**풋마늘:** 마늘이 아직 덜 자란 상태의 어린 마늘로, 부추와 비슷한 식감이에요.

**활용:** 마늘쫑볶음, 마늘쫑장아찌, 풋마늘무침, 풋마늘찜

## 6월 제철 과일

### 참외

6월이 되면 본격적으로 참외 시즌이 시작돼요. 한국 참외는 세계적으로도 독특한 품종으로, 6~8월이 제철이에요.

**고르는 법:**
- 노란색이 선명하고 흰 줄이 뚜렷한 것
- 꼭지가 마르지 않고 신선한 것
- 들었을 때 묵직한 것
- 향이 달콤하게 나는 것

**영양:** 비타민 C, 엽산, 칼륨 풍부. 특히 씨 주변의 태좌(하얀 부분)에 폴리아민이 풍부해 세포 재생에 도움.

**활용:** 생과일, 참외화채, 참외 냉국, 참외 피클

### 복분자

6월 중순~7월 초가 복분자의 제철이에요. 검붉은 색에 새콤달콤한 맛이 특징이에요.

**고르는 법:**
- 알이 굵고 빽빽하게 달린 것
- 색이 균일하게 검붉은 것
- 물기가 없고 신선한 것

**영양:** 안토시아닌, 비타민 C, 엘라직산 등 항산화 성분이 풍부해요. 눈 건강과 피로 회복에 좋다고 알려져 있어요.

**활용:** 생과일, 복분자주, 복분자청, 주스, 요거트 토핑

### 앵두

6월 초~중순이 앵두 제철이에요. 새콤달콤하고 알이 작아 장식 과일로도 예뻐요.

**고르는 법:**
- 선명한 빨간색
- 꼭지가 싱싱한 것
- 과육이 단단한 것

**활용:** 앵두화채, 앵두청, 앵두잼

### 매실 (황매실 시기)

6월 말에는 황매실이 익어요. 단맛이 강하고 향이 풍부한 황매실은 청보다 잼이나 음료에 더 잘 어울려요.

## 6월 제철 해산물

### 오징어

6월부터 동해안 오징어가 본격적으로 시즌을 맞이해요. 여름 오징어는 살이 쫄깃하고 맛이 진해요.

**고르는 법:**
- 몸통이 투명하거나 살짝 보랏빛이 도는 것 (신선도 높음)
- 내장이 터지지 않은 것
- 눈이 투명하고 맑은 것
- 냄새가 은은하게 바다 향인 것

**활용:** 오징어볶음, 오징어무침, 오징어순대, 오징어젓갈

### 갈치

여름 갈치는 살이 통통하고 기름져서 맛이 좋아요. 6~9월이 제철이에요.

**고르는 법:**
- 은빛 비늘이 고루 남아 있는 것
- 눈이 맑고 투명한 것
- 냄새가 심하지 않은 것
- 몸통이 두텁고 묵직한 것

**활용:** 갈치조림, 갈치구이, 갈치국

### 병어

6~8월이 병어의 제철이에요. 살이 부드럽고 담백해서 구이나 조림으로 즐기기 좋아요.

**고르는 법:**
- 은빛이 선명하고 광택이 있는 것
- 눌렀을 때 탄력이 있는 것
- 눈이 맑은 것

**활용:** 병어조림, 병어구이, 병어회

### 꽃게

서해안 꽃게는 봄(4~6월)과 가을(9~11월)이 제철이에요. 6월은 봄 꽃게 시즌의 마무리 시기로, 알이 꽉 찬 암게를 저렴하게 구할 수 있어요.

**고르는 법:**
- 배딱지가 꽉 차 있는 것
- 다리와 집게가 모두 붙어 있는 것
- 살아서 움직이는 것

**활용:** 꽃게탕, 꽃게찜, 간장게장, 양념게장

## 6월 식재료 보관법 요약

- **매실:** 구입 즉시 청으로 담그는 게 가장 좋음. 보관이 필요하면 냉장 3~5일, 냉동 가능
- **햇감자:** 서늘하고 어두운 곳 상온 보관 (냉장 비권장)
- **참외:** 덜 익은 것은 상온 후숙, 완숙은 냉장 3~5일
- **복분자:** 냉장 1~2일, 냉동 6개월
- **오징어:** 즉시 손질 후 냉동 보관 권장

## 마무리

6월은 매실, 참외, 복분자, 햇감자, 오징어 등 계절의 맛을 느낄 수 있는 풍성한 달이에요. 제철 식재료는 맛과 영양이 최고조에 달하고 가격도 합리적이에요. 지금 이 계절에만 먹을 수 있는 식재료들을 충분히 즐겨보세요. 특히 매실은 제철이 짧으니, 나오는 시기에 놓치지 말고 청이나 주로 담가두면 1년 내내 활용할 수 있어요!
