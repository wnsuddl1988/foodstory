장마가 끝나고 본격적인 여름이 시작되는 7월. 덥고 습하지만 제철 식재료만큼은 풍성하고 맛있어요. 수박이 당도를 자랑하고, 복숭아가 달콤하게 익어가며, 민어와 전복이 제 맛을 내는 7월. 지금 먹어야 가장 맛있는 식재료들을 총정리해드릴게요.

## 7월 제철 채소

### 옥수수

7~8월이 국내산 찰옥수수의 제철이에요. 갓 수확한 찰옥수수는 알이 탱탱하고 단맛이 풍부해요.

**고르는 법:**
- 껍질이 선명한 녹색이고 싱싱한 것
- 수염(옥수수 수염)이 갈색이고 촉촉한 것 (노란 수염은 덜 익은 것)
- 낱알이 고르고 빽빽하게 박힌 것
- 눌렀을 때 살짝 탄력이 있는 것

**보관법:**
- 껍질째 냉장: 2~3일
- 삶은 후 냉동: 약 1년 (알만 뜯어서 냉동하면 더 편리)

**활용:** 옥수수구이, 옥수수수프, 콘 샐러드, 옥수수밥

### 가지

가지는 7~9월이 제철이에요. 이 시기에 가지는 과육이 탱탱하고 단맛이 강해요.

**고르는 법:**
- 껍질에 윤기가 있고 진한 보라색인 것
- 꼭지가 싱싱하고 날카로운 가시가 있는 것 (신선 지표)
- 손으로 들었을 때 묵직한 것
- 주름이나 흠집이 없는 것

**보관법:**
- 키친타월로 감싸 냉장: 5~7일
- 길게 잘라 냉동: 2~3개월 (볶음, 찌개용)

**활용:** 가지볶음, 가지나물, 가지전, 가지구이

### 오크라

여름 수입 채소로 최근 국내에서도 재배가 늘고 있어요. 7~9월이 제철이에요.

**특징:** 특유의 점액질 성분(뮤신)이 위벽을 보호하고 혈당 조절에 도움을 줘요.

**고르는 법:**
- 선명한 초록색
- 크기가 너무 크지 않은 것 (7~8cm 정도가 부드러움)
- 표면에 솜털이 있는 것

**활용:** 살짝 데쳐 샐러드, 튀김, 볶음, 카레에 넣기

### 풋콩·에다마메

여름이 제철인 풋콩은 단백질과 이소플라본이 풍부해요.

**고르는 법:**
- 껍질이 선명한 녹색
- 알이 통통하게 찬 것
- 수염 끝이 갈색인 것

**보관법:**
- 냉장: 2~3일 (빨리 당분이 전분으로 변해 맛 감소)
- 삶아서 냉동: 1~2개월

**활용:** 삶아서 소금과 함께 간식, 밥에 넣기, 두부 만들기

## 7월 제철 과일

### 수박

수박은 6~8월이 제철인데, 7월 중순~8월 초가 당도가 가장 높은 시기예요.

**고르는 법:**
- 두드렸을 때 '통통' 울리는 소리
- 배꼽 부분이 작고 둥근 것
- 줄무늬가 선명한 것
- 들었을 때 묵직한 것

**영양:** 수분이 91%, 리코펜이 풍부해 항산화 효과. 아르기닌 성분이 혈관 건강에 좋아요.

### 복숭아

7~8월이 복숭아 제철이에요. 종류에 따라 시기가 다른데, 7월은 천도복숭아와 일반 복숭아가 본격적으로 나오는 시기예요.

**고르는 법:**
- 향이 진하게 나는 것
- 살짝 눌렀을 때 적당히 들어오는 것
- 껍질색이 고르고 흠집이 없는 것

### 참외

6~7월이 참외 최성수기예요. 7월 참외는 당도가 가장 높은 시기예요.

### 블루베리

7~8월이 국내산 블루베리 제철이에요. 안토시아닌이 풍부해 눈 건강에 특히 좋아요.

**고르는 법:**
- 알이 굵고 파란색이 균일한 것
- 표면에 흰 분(bloom)이 덮인 것 (신선)
- 단단하고 주름이 없는 것

**보관법:**
- 씻지 않고 냉장: 1~2주
- 냉동: 6개월~1년 (안토시아닌 보존)

## 7월 제철 해산물

### 민어

7~8월이 민어 제철이에요. 여름 보양식의 대표 식재료로, 살이 담백하고 단백질이 풍부해요.

**고르는 법:**
- 눈이 맑고 돌출된 것
- 아가미가 선명한 빨간색
- 비늘이 촘촘하게 붙어 있는 것
- 냄새가 은은하게 바다향인 것

**활용:** 민어회, 민어매운탕, 민어구이, 민어전

### 전복

여름 전복은 미역을 먹고 자라 살이 통통하고 맛이 진해요. 7~8월이 산란기 전으로 맛이 절정이에요.

**고르는 법:**
- 껍데기에 붙은 살이 팽팽하고 탄력 있는 것
- 살아서 움직이는 것
- 껍데기에 광택이 있는 것

**활용:** 전복죽, 전복찜, 전복회, 전복버터구이

### 새우

7~9월은 새우가 살찌는 시기예요. 특히 제주산 차새우, 서해안 흰다리새우가 제철이에요.

**고르는 법:**
- 껍질이 투명하고 광택 있는 것
- 머리가 떨어지지 않고 붙어 있는 것
- 냄새가 바다향인 것

**활용:** 새우구이, 새우튀김, 새우볶음밥, 새우탕

### 장어 (민물장어·붕장어)

삼복더위 보양식으로 유명한 장어. 7~8월이 제철이에요.

**고르는 법:**
- 살아서 힘차게 움직이는 것
- 껍질에 윤기가 있는 것
- 배 부분이 황금빛인 것

**활용:** 장어구이, 장어덮밥, 장어탕, 민물장어찜

## 7월 제철 식재료 보관 요약

| 식재료 | 보관 방법 |
|---|---|
| 옥수수 | 껍질째 냉장 2~3일 / 삶아서 냉동 1년 |
| 가지 | 키친타월 포장 냉장 5~7일 |
| 수박 | 자른 후 랩 밀착 냉장 3~5일 |
| 복숭아 | 덜 익으면 상온 후숙 → 냉장 3~5일 |
| 블루베리 | 씻지 않고 냉장 1~2주 / 냉동 6개월 |
| 전복 | 살아있는 상태 구매 후 즉시 조리 |
| 새우 | 즉시 손질 냉동 (6개월) |

## 마무리

7월은 무더운 여름이지만 수박, 복숭아, 옥수수, 블루베리 등 달콤하고 시원한 과일과 채소, 보양에 좋은 민어, 전복, 장어가 풍성한 달이에요. 제철 식재료는 영양이 풍부하고 가격도 합리적이니 지금 이 계절을 놓치지 마세요. 여름 보양식도 챙겨가면서 이 계절을 건강하게 나보세요!
